# Serophero News Portal (WordPress)

This repository contains a WordPress-based news portal site.

## Local development (Laragon)

- Place the project in your Laragon `www` directory.
- Create a virtual host (e.g. `newsportallocal` / `newsportallocal.test`).
- Ensure the database exists and `wp-config.php` matches your local DB settings.

## Notes

- WordPress core, themes, and plugins are included in this repo as-is.
